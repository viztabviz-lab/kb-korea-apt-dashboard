### 2-4. 시도 → 시군구 드릴다운
- **전국 선택 시**: 기존과 동일(시도별 비교).
- **특정 시도 선택 시**: 해당 시도의 **시군구 기준**으로 전환.
  - 상단 카드 / 추이 그래프: 선택 시도 기준
  - "가격지수 비교" 패널 → **시군구별 KB vs 부동산원**
  - "평균가격 비교" 패널 → **시군구별 전세가율(KB)** 로 교체
    (시군구 평균가는 두 기관 모두 미제공)
- 시군구 데이터는 시도별 파일로 분리해 **선택 시 lazy-load** (초기 로딩 경량화).
  - `data/sg/kb/<slug>.json` — 매매/전세 가격지수 + 전세가율
  - `data/sg/reb/<slug>.json` — 매매/전세 가격지수
- 차트 높이는 시군구 개수에 맞춰 자동 조정.

## 3. 데이터 소스별 제공 범위 (중요)

### 3-1. 지표 × 단위별 제공 여부

| 지표 | 시도 (KB/REB) | 시군구 (KB) | 시군구 (REB) |
|---|---|---|---|
| 매매·전세 **가격지수** | ✅ / ✅ | ✅ | ✅ |
| 매매·전세 **평균가격** | ✅ / ✅ | ❌ (API 미제공) | ❌ (API 미제공) |
| **전세가율** | ✅ / ❌(미공표) | ✅ | ❌ (미공표) |

> - 평균가격은 **두 기관 모두 시도 단위까지만** 제공.
> - 전세가율은 **부동산원이 OpenAPI로 공표하지 않음** → KB만.

### 3-2. 기간(시계열) 범위 — KB 주간 지수의 한계

| 지표 | KB | 부동산원 |
|---|---|---|
| 가격지수 (주간) | **2024-06-24 ~** (약 2년, 101주) | 2012-05-07 ~ (735주) |
| 평균가격 (월간) | 2008~2011 ~ (장기) | 2012-01 ~ |

- KB 공개 API(`priceIndex`)는 **주간 가격지수를 최근 약 2년치만** 반환(`날짜리스트` 101개 고정).
  2024년 리베이스 기준시점 이전 주간 지수는 이 엔드포인트에 없음.
- KB도 **월간 평균가격은 2008년부터** 장기 제공.
- 가격지수 추이 차트는 공통 시작일=100 재지수화 → 실제 겹치는 구간은 2024-06부터.

## 4. API 처리에서 알게 된 점 (재현/유지보수용)

### KB부동산 데이터허브 (`data-api.kbland.kr`, 인증키 불필요)
- 최상위 호출(지역코드 미지정) → `전국` + 광역집계(`강북14개구` 등) + **17개 시도** 일괄 반환.
- 시군구 드릴다운: `지역코드=<시도코드>` 로 호출 → 해당 시도의 시군구 반환
  (`경기` 는 `수원시` + `수원시 장안구` 처럼 상위 시·하위 구 모두 포함, 55개).
- **주의**: `avgPrc`(평균가격)은 `지역코드` 를 무시하고 항상 전국 목록을 반환
  → 시군구 평균가격은 수집 불가.

### 한국부동산원 R-ONE (`reb.or.kr/r-one/openapi`, 인증키 필요)
- `REB_API_KEY` 는 GitHub Actions Secret 으로 등록되어 있어 워크플로에서만 수집 가능
  (로컬엔 키 미설정 → 로컬 실행 불가, 대시보드는 부동산원 데이터 없어도 KB만 렌더).
- 시도 식별: `CLS_FULLNM` 이 단일 지역명(공백·`>` 없음)인 행.
- **시군구 식별**: `CLS_FULLNM` 은 `>` 로 구분된 계층 경로.
  - 예) `경기>경부1권>과천시`, `충북>청주시>상당구`, `서울>강북지역>마포구`
  - 중간에 **권역(`경부1권`)·지역(`강북지역`) 묶음 레벨**이 낌.
  - 표시명은 **행정단위(…시/군/구) 토큰만** 남겨 KB 명명과 일치시킴
    → `마포구`, `청주시 상당구`, `수원시 장안구` (집계행 `강남지역` 등은 제외).
- 인증키 없이 호출하면 표당 5건 샘플만 반환(필드 구조 확인용으로 활용).
